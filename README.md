# 🌱 Sostenibilità e innovazione in Bonifiche Ferraresi

## 🎓 Project Work - Unipegaso

**Corso:** Informatica per le Aziende Digitali (L-31)  
**Anno accademico:** 2025 / 2026

## 📖 Descrizione del progetto
Questo progetto ha l’obiettivo di analizzare e raccontare l’impegno di **Bonifiche Ferraresi** nel campo della sostenibilità, con particolare attenzione all’innovazione in ambito agricolo e alla gestione responsabile delle risorse.

## 🎯 Obiettivi
- Analizzare le pratiche di sostenibilità aziendale  
- Evidenziare l’innovazione tecnologica nel settore agricolo  
- Comprendere l’impatto ambientale, economico e sociale
